Zikpad

Zikpad is an open-source software to quickly write music, that is intended to be simple. Do not hesitate to like the project on github.

Lilypond

Zikpad produces Lilypond file and thus enables to produce high-quality scores.

Already implemented features

  • You add notes by clicking.
  • You can listen to your music!
  • You add a rest by clicking on a note
  • You can move all notes
  • Durations of notes are computed from the context
  • The color palettes are the different voices
  • Load and save work in the offline software
